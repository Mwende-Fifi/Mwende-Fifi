## 🚀 About Me 
I analyze transaction and user data to find where fraud, risk, and revenue move, then build the pipelines and dashboards that let teams act on it.
At Hurupay, I built anomaly detection scripts on live payment data, wrote multi-table SQL across MongoDB and PostgreSQL for fraud workflows and BI reporting, pulled from open banking APIs to enrich datasets, and paired Amplitude and Intercom with raw data to connect user behavior to the numbers.
Python (pandas, scikit-learn, XGBoost), R, SQL, Tableau, and Excel. My credit risk project involved feature engineering (loan-to-income ratio, interest burden) and classifier benchmarking with cross-validation.
BSc Statistics, JKUAT. Former Data Science TA — supported 50+ students, contributed to a 15% average improvement in project performance.
Open to Data Analyst, Junior Data Scientist, or Risk/Fraud Analyst roles, particularly in fintech.
